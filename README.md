# Python Pizza Delivery Calculator

This is a simple command-line Python program that calculates the total cost of a pizza order based on the pizza size and selected toppings.

## Features

- Choose pizza size (Small, Medium, Large)
- Option to add pepperoni
- Option to add extra cheese
- Automatically calculates the final bill

## Pricing Rules

| Item | Price |
|-----|------|
| Small Pizza | $15 |
| Medium Pizza | $20 |
| Large Pizza | $25 |
| Pepperoni (Small) | $2 |
| Pepperoni (Medium/Large) | $3 |
| Extra Cheese | $1 |

## How It Works

1. The program asks the user for the pizza size.
2. The user selects whether they want pepperoni.
3. The user chooses if they want extra cheese.
4. The program calculates and prints the final bill.


## How to Run

Make sure Python is installed, then run:

```bash
python pizza_delivery.py
